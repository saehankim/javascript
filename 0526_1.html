자바스크립트 오류처리

try catch 같이 사용한다.

try 블록에 오류를 테스트할 구문을 작성

catch 블록에 오류를 처리할 구문을 작성 

        try{
        }catch(err){
        }

1~10 사이의 숫자를 입력 

    try{
        // 실제작성할 테스트 코드블록
    }
    catch(err){
        // 에러 코드블록
    }
    finally{
        // try catch 상관없이 실행되는 코드블록
    }
    
    자바스크립트 에러객체
    오류발생시 오류정보를 제공하는
    오류관련 객체가 내장되어있다.
    
    try catch 에서 catch(err) err-> Error객체
    err.name   오류 이름
    err.message   오류 메시지

    에러이름 종류
    1) EvalError
    2) RangeError
    3) ReferenceError
    4) SyntaxError
    5) TypeError
    6) URIError
